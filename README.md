# HOW TO FLASH
1. get urself some Samsung drivers, any vbmeta.tar FOR A02 SPECIFICALLY, and Odin.
2. get into download mode, put recovery.tar into AP, AND TURN OFF AUTO REBOOT.
3. flash. reset. then reconnect ur phone. DO NOT TURN IT OFF
4. TURN OFF AUTO REBOOT, put the vbmeta.tar into the AP, and flash it
5. reset. leave the download mode and instantly get into... for now just flashing backlight of the phone, I'm still working on recovery :DD
